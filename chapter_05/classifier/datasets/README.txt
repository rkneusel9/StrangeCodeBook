Datasets for use with classify.sno
----------------------------------

All of these datasets are available on the UCI Machine Learning Repository
and are freely distributable.

Irises (https://archive.ics.uci.edu/ml/datasets/iris)
    This is the classic three class irises dataset by Fisher.

Banknote (https://archive.ics.uci.edu/ml/datasets/banknote+authentication)
    A dataset for detecting fake banknotes.

Cancer (https://archive.ics.uci.edu/ml/datasets/Breast+Cancer+Wisconsin+%28Diagnostic%29)
    Histology slide data for breast cancer diagnosis.

Ecoli (https://archive.ics.uci.edu/ml/datasets/Ecoli)
    Types of E. coli bacteria.

Haberman (https://archive.ics.uci.edu/ml/datasets/Haberman%27s+Survival)
    Three features, breast cancer survival

Seeds (https://archive.ics.uci.edu/ml/datasets/seeds)
    Seed features for species identification.  Three classes.

Wine (https://archive.ics.uci.edu/ml/datasets/wine)
    Wine features to classify wine origin.
